# Education

一组给小学低年级孩子的教育互动小项目。全部是**单文件网页**：不用装 App、不用联网、
平板或电脑浏览器打开即玩。欢迎自取自用。

## 项目列表

### [abc-phonics](abc-phonics/) — ABC 发音乐园

自然拼读（Phonics）互动游戏，帮刚学英语的孩子建立"字母 → 发音"的对应规则
（比如同一个字母 a，在 apple 和 cake 里发音不同）。

- **内容**：26 个字母 + 17 组常见组合音（sh / ch / th / ee / oo / igh …），
  每种发音配单音示范和 3~4 个高频词（带表情图）；th 含清浊两音（three vs this）。
- **玩法**：
  - 探索模式：翻卡片听发音、跟读单词，目标字母高亮显示，每个词有 🐢 慢速朗读；
  - 三个闯关游戏：听音选词 / 找发音 / 辨音双胞胎（专练 m·n、b·d、b·p、f·v 易混音）；
  - 奖励系统：答对得 ⭐，集星解锁贴纸，字母答对点亮小皇冠；
  - 家长面板：答题量、正确率、高频错词一目了然。
- **语音**：442 条内嵌语音，完全离线。**每个字母/组合的单音示范是美国教师的
  真人教学录音**（来源 Sound City Reading，免费教育用途授权，详见
  `abc-phonics/recordings/CREDITS.md`；本项目非商业，如需商用须先取得原作者许可）；
  单词朗读为美音合成（常速+慢速）+ 中文引导语。全部示范音经过双引擎机器听审
  （faster-whisper large-v3 + allosaurus，见 `tools/qa_gate.py`）。

#### 怎么玩（家长看这里）

1. 下载仓库（绿色 `Code` 按钮 → `Download ZIP`），解压；
2. 找到 `abc-phonics/dist/index.html` 这**一个文件**，双击用浏览器打开即可；
3. 平板使用：把这个文件传到平板（微信 / 邮件都行），用浏览器打开，
   建议"添加到主屏幕"，就像装了个小 App。孩子的星星进度自动保存在本机。

#### 怎么定制（会一点技术的看这里）

仅需 macOS（语音由系统自带的 `say` 合成）：

```bash
cd abc-phonics
# 1. 改 data.py：词表、发音示范、易混音对、奖励贴纸都在这一个文件里
# 2. 重建（语音有缓存，增量很快）：
python3 build.py
# 3. 成品在 dist/index.html
```

发音示范的拟声词字典经过语音识别"机器试听"校准（详见 `data.py` 头注释）；
如果觉得某个音不准，改对应 `demo` 字段重建即可，也可以用
`tools/audit_demos.py`（需 `pip3 install faster-whisper`）机器复听。
